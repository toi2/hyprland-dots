# hyprland-dots

My personal, modular Hyprland configuration files for Arch and Arch-based os like EndeavourOS, ManjaroOS, CachyOS. 

## Features
* **Language:** Structured entirely in Lua (`.lua`) instead of standard configuration syntax for better modularity.
* **Shell Integration:** Tailored specifically to run alongside and support the **Caelestia shell** setup.
* **Distribution:** Optimized for Arch-based OS.

## Structure
* `hyprland.lua` - Main initialization file.
* `variables.lua` & `hypr-vars.lua` - Environment and system variables.
* `monitors.conf` & `workspaces.conf` - Display and workspace management.
* `scheme/` - Color schemes and appearance configurations.
